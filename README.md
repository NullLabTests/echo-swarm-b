# Echo Swarm

A recursive self-improving multi-agent system built with CrewAI and Groq.

### Description
This project runs a persistent swarm of agents that research recent advances in self-improving AI, propose code changes, evaluate them, and learn from each loop using trajectory memory.

### Current Version
v0.2 - HyperReflector enabled

### Setup
1. Run `./setup.sh`
2. Add your Groq API key to `.env`
3. Run `./run_echo.sh`
